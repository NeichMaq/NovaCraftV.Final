Ayuda

Informacion basica: Si aun no entiendes cual es la mecanica o el modo de juego del servidor esta informacion te servira:
Basicamente es una mezcla entre un servidor survival normal (con plugins para mejorar la experiencia de juego) y un server RPG.
Lorenica es el mundo que ocupa esa parte, lleno de mobs especiales que dropean items que sirven como moneda de intercambio en el servidor, tales como las Monedas de Oro, Chaos, etc.
Ayuda util:Si tienes preguntas basicas como "¿Como protejo mi casa?" o "¿Cuales son los telenstransportes del servidor?" puedes usar /Ayuda para mas detalles.

Usa /warps para el menu de telenstransportes.
Si te quedaste con dudas de como funciona Lorencia puedes cisitar el warp o usar el comando /Lorencia.
Si se necesita mas ayuda, preguntar por el discord o por el mismo chat del juego.
